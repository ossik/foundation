Companies can be members of the Foundation to help assure the future of Robot Framework and be part of something greater. Join us!

What we do

The Foundation will provide bug fixes, review and accept Pull Requests and make Robot Framework releases. We will also be the source of Robot Framework knowledge. We keep the Robot Framework documentation updated, have a list of projects where Robot Framework is used as references and make sure that there is a base to communicate with the community.
